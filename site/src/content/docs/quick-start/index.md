`gopls-mcp` empowers your code agent with **Semantic Precision**, **Zero-RTT Context**, and **Token Efficiency**.

### Installation

**Linux / macOS:**
```bash
curl -sSL https://gopls-mcp.org/install.sh | bash
```

**Windows (PowerShell):**
```powershell
irm https://gopls-mcp.org/install.ps1 | iex
```
